目的

Gyao は人気のある動画配信サイトのひとつです。

しかし、このサイトは Google Chrome に対応していません。 Google Chrome でアクセスすると通常は以下のような文面が表示されるのみです。

お使いのブラウザでは、映像を視聴できません。

GyaO!の映像を見るために必要な環境については「視聴環境の確認」をご覧ください。
お使いの環境の判定と解決方法については、利用環境判定ページをご覧ください。

この拡張は Gyao の動画を視聴することを支援するものです。

使い方

実際の画面で説明します。

まず個別の動画ページのタイトルの上に、動画へダイレクトに繋げる URL が表示されるようになります。

シリーズ物であれば、一覧ページでの各タイトルの下にも表示されます。 (表示モードを「詳細表示モード」にしているときだけです。 「画像とタイトル」に設定されているときは表示されません。)

表示された URL を Windows Media Pleyer 等の mms プロトコルに対応したプレイヤに入力すればそのまま視聴することが出来ます。

Windows では単にクリックすれば Windows Media Player が起動するでしょう。

注意点

  • Gyao の動画データは DRM で保護されています。 DRM 付きの WMV 形式動画を再生するプレイヤはおそらく Windows にしかありません。 また、ダウンローダで手元に保存していても一定期間後には再生できなくなります。
  • この拡張の古いバージョンではローカルの時計の時刻があっていることが必要でした。 現在のバージョンではサーバからの返答を元に時刻を取得しているので、時刻設定の影響を受けません。
  • この拡張は作成時点における Gyao のサイト構成を独自に調査して作成した非公式なものです。 Gyao がサイトに変更を加える等した場合にはこの拡張は機能しなくなる可能性があり得ます。
  • この拡張は Gyao やその運営企業各位とは全く関係のない非公式なものです。 この拡張について Gyao に問い合わせることはしないで下さい。
  • あなたがこの拡張を入手することで得たのはこの拡張それのみであり、保証やサポートを一切含みません。
